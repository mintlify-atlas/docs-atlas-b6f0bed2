## What is Deni AI?

Deni AI is a modern AI chat platform designed to make advanced AI accessible to everyone, especially those who can't afford premium AI services. Access multiple cutting-edge AI providers including OpenAI, Anthropic, Google, Groq, xAI, and OpenRouter through a clean, intuitive interface - all for free.

Whether you're a developer, student, creative professional, or just curious about AI, Deni AI provides you with the tools to chat with the latest AI models without breaking the bank.

## Technology stack

Deni AI is built with modern web technologies:

- **Framework**: Next.js 16 with App Router and React 19
- **Language**: TypeScript (strict mode)
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: better-auth with OAuth, email/password, passkeys, and 2FA
- **Styling**: Tailwind CSS v4 with shadcn/ui components
- **AI SDK**: Vercel AI SDK with multi-provider support
- **Runtime**: Bun (recommended) or Node.js 20+
