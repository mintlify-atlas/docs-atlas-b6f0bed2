## Switching Between Models

### In-Chat Model Selection

Change models at any time during a conversation:

<Steps>
  <Step title="Open Model Selector">
    Click the model selector button in the chat composer (bottom of the chat interface).
  </Step>
  <Step title="Browse Models">
    View all available models organized by provider. Models display their features as badges (Reasoning, Smart, Fast, etc.).
  </Step>
  <Step title="Select Model">
    Click on any model to switch. Your next message will use the selected model.
  </Step>
</Steps>

<Info>
You can switch models mid-conversation. Each message is processed by the model selected at the time of sending.
</Info>

### Model Selection UI

The model selector provides rich information:

- **Model name**: Clear identification of each model
- **Provider badge**: Shows the AI company (OpenAI, Anthropic, Google, xAI)
- **Feature badges**: Visual indicators of model capabilities
  - Reasoning: Advanced logical thinking
  - Smart: High intelligence
  - Fast: Quick response times
  - Coding: Programming expertise
  - Smartest/Fastest: Top-tier in category
- **Premium indicator**: Shows which models require premium access
- **Custom badge**: Identifies user-created custom models

## Model Availability

<Info>
Model availability depends on your subscription tier and usage limits. Some models require premium access or additional configuration.
</Info>

### Free Tier
Access to select OpenAI, Google, xAI, and OpenRouter models with usage limits.

### Premium Tier
Unlimited access to all models including Claude (Anthropic) with higher usage limits.

### BYOK (Bring Your Own Key)
Use your own API keys to access models directly, bypassing usage limits.

See [Custom Models](/features/custom-models) for information on adding your own models and API keys.
